# Add Phase Notes (Reflexion)

Add a Reflexion entry to @Documents/PHASE_NOTES.md after completing a phase.

**Instructions:**
1. Create or append to Documents/PHASE_NOTES.md
2. Use this template:

```markdown
## Phase N — [Name] — [Date]

### Completed
- [x] Task X.Y
- [x] Task X.Z

### Lessons Learned
- [What went wrong and how we fixed it]
- [What to do differently next time]

### Blockers Resolved
- [Any blockers and resolutions]
```

3. Fill in based on the phase just completed
4. Reference @Documents/IMPLEMENTATION_PLAN.md for task numbers

**User should specify:** Phase number (1–6) and date when invoking.

# Verify Phase Completion

Launch the verifier agent to validate that a phase is complete.

**Instructions:**
1. Identify what was claimed to be completed (phase number or specific tasks)
2. Check that the implementation exists and is functional
3. Run relevant tests (`npm test` or `jest`)
4. Verify against @Documents/IMPLEMENTATION_PLAN.md phase checklist
5. Look for edge cases that may have been missed

**Report format:**
- **Verified and passed:** What works as expected
- **Incomplete or broken:** What was claimed but does not work
- **Issues to address:** Specific fixes needed with file paths and line references

**User should specify:** Phase number (1–6) in the chat when invoking.
